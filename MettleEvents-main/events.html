Coding Related Problem in Meta

A novel competition to test metallurgical reasoning and data analytics skills among the participants. We pose challenges to participants and provide them with data to come up with models based on machine learning algorithms. The event is hosted on Kaggle and candidate with model with least error emerge as winners.
